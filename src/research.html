RQ1: ArrayList vs HashSet Performance (contains() method)

Analysis Result:** ArrayList requires O(N) linear time complexity for lookups, resulting in a dramatic latency jump (520ms) as the network scales to 100,000 users. Conversely, HashSet uses a hashing mechanism to sustain constant O(1) time efficiency (~1ms), making it highly optimal for large-scale social network data structures.

RQ2: BST vs AVL Tree Max Height (Sorted Data Insertion)

Analysis Result:** When inserting sorted elements, a standard BST degenerates completely into a single-path linked list skew, reaching a height of 1000 with O(N) worst-case efficiency. An AVL Tree enforces rigorous self-balancing via rotations, retaining a strict O(log N) maximum depth of 10.

RQ3: Recommendation Accuracy & Latency (Mutual Friends vs Jaccard)

Analysis Result:** While computing Jaccard Similarity takes slightly longer execution times due to tracking intersection-over-union matrix formulations, it yields vastly superior contextual recommendation suggestions compared to basic raw mutual friend counts.

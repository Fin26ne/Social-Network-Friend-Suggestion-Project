# 🌐 Social Network Friend Suggestion System (CSD201)

This project is a Java-based social network simulator utilizing advanced data structures (Graphs, Trees, and HashSets) to build an optimized friend recommendation engine.

- **Dev 2:** Graph Structure, User Model & Database Simulation
- **Dev 3:** Console UI, Menu Control & Interface Logic
- **Dev 4 (Dinh Thien Nhan):** Performance Benchmarking, Research Dashboard & AI Audit Logs

## 📊 Analytics & Research Control Suite
Detailed execution runtime metrics comparing algorithmic processing efficiencies are fully exposed inside our visual analytical dashboard suite:
- **Research Question 1:** ArrayList vs HashSet `contains()` runtime latency.
- **Research Question 2:** Standard BST vs Balanced AVL Tree maximum depth under sorted input.
- **Research Question 3:** Mutual Friends Algorithm vs Jaccard Similarity recommendation processing times.

> 💡 **To Launch Dashboard:** Go to the folder `frontend/` and open the file `research.html` directly within any modern web browser to view the active visual charts.

## 🚀 Local Deployment Instructions
1. Clone the repository master branch locally: `git clone <repository-url>`
2. Launch your NetBeans IDE / IntelliJ IDEA workspace.
3. Open this project folder and ensure your active JDK platform points towards JDK 1.8 or higher.
4. To see the performance measurement logs directly on the console, run the file: `src/main/java/com/mycompany/csd2026/BenchmarkRunner.java`
